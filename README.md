# Simple blog in GatsbyJS
<br/>

> ..Simplicity is always perfect.

It's very simple blog created in GatsbyJS.

  
<br>

#### Technologies:
  - HTML5
  - SASS (Scss)
  - JavaScript ES6
  - GatsbyJS
  - DatoCMS


### Installation

Install the dependencies and start the server.

```sh
$ npm install
$ gatsby develop
```
